HackHunt 🚀

HackHunt is a full-stack web application designed to help users discover and track hackathons from multiple platforms in one place.

🔥 Features

- 🔍 Browse hackathons from different platforms
- 📊 Filter hackathons based on preferences
- 🤖 Automated scraping of hackathon data
- 📡 Real-time updates
- 🔐 User authentication system

🛠 Tech Stack

- Frontend: React.js, CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Other Tools: Vite, APIs, Web Scraping

📁 Project Structure

- "/frontend" → User interface
- "/backend" → Scraping & API handling
- "/server" → Core backend logic and database

⚙️ Installation & Setup

1. Clone the repository:

git clone https://github.com/your-username/HackHunt.git

2. Install dependencies:

npm install
cd hackathon-tracker/backend && npm install
cd ../../server && npm install

3. Run the project:

npm run dev
